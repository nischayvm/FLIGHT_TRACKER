# Flight Tracker

A real-time flight tracking application built with the MERN stack (MongoDB, Express, React, Node.js).

## Features
- Real-time flight tracking using OpenSky Network data
- Interactive map interface with custom plane icons
- Flight details sidebar with stats and search
- Dark mode UI with neon accents

## Tech Stack
- **Frontend**: React, Vite, Leaflet, Socket.io-client
- **Backend**: Node.js, Express, Socket.io, MongoDB
